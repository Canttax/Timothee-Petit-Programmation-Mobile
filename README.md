# Timothee-Petit-Programmation-Mobile

This repository contains all my practical work (TP - Travaux Pratiques) for Mobile Programming courses.

## Repository Structure

Each TP is organized in its own directory with the following structure:

```
TP[number]-[name]/
├── README.md          # TP description and instructions
├── src/              # Source code
├── docs/             # Documentation
└── assets/           # Resources (images, data files, etc.)
```
